# Retirement Simulator

The retirement simulator is a standalone web app hosted at
**[nbellowe.github.io/retirement-simulator](https://nbellowe.github.io/retirement-simulator)**.

It runs a Monte Carlo simulation entirely in your browser — no server, no credentials.

---

## Quick Start

```bash
# Generate a starter profile in your current directory
monarch init-profile

# Edit it with your numbers
open profile.yaml
```

Then visit the simulator, drag in your `profile.yaml`, and explore.

---

## Profile loading

- **Drag-and-drop** `profile.yaml` anywhere on the simulator page
- **Load file** button in the YAML Settings panel
- Paste YAML directly into the YAML editor and click **Apply YAML**

Your profile is saved in `localStorage` and restores automatically on next visit.
